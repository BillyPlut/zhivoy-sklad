Чтобы данные были доступны с телефона и компьютера, нужно подключить бесплатную базу данных Firebase (Google). Это займёт ~5 минут один раз.
1
Зайдите на console.firebase.google.com через Google-аккаунт и создайте новый проект
2
В проекте откройте Firestore Database → «Создать базу данных» → режим test mode
3
Откройте Authentication → «Начать» → включите провайдер Email/Password
4
В настройках проекта (⚙️) → Ваши приложения → добавьте Web-приложение → скопируйте firebaseConfig (JSON) и вставьте сюда:
